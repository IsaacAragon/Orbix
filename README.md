## Descripción del proyecto
**Orbix** una aplicación que moderniza el mercado de renta de vehículos en Nicaragua.
Nuestra plataforma conecta a propietarios de vehículos con usuarios que buscan alquilar, así simplificando el proceso de renta. Fácil, seguro y desde cualquier lugar.

## Tecnologías usadas
- **IDE:** Android Studio
- **UI Framework:** Jetpack Compose
- **Lenguaje:** Kotlin
- **API**: 24 (Android "Nougat" 7.0 y superiores)

## Instrucciones para ejecutar la app

1. Clone el repositorio
	```bash
	git clone https://github.com/IsaacAragon/Orbix.git
	```
2. Abrir el proyecto en Android Studio
- En la pestaña **File** seleccione **Open**
- Busque el proyecto que acaba de clonar
- Espere a que Gradle sincronice las dependencias

3. Ejecutar la aplicación
- Prepare el emulador o conecte un dispositivo físico Android
- Presione el botón **Run**
- Espere a que el proyecto compile y se abra en si dispositivo
